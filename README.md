# DODO PDF Reader

![dodo Logo](./logo/logo.png)

A fast and minimal PDF reader.




## Features

- Caching and pre-fetching pages for faster page rendering
- Vim-like keybindings
- configured using TOML language
- Faster search

## Dependencies

- Qt6
- Poppler
- Poppler-Qt6
