# Dodo
A minimalistic PDF Viewer written in C++ using `Qt` and `Poppler` that uses `VIM` keybindings (mostly)

# Why ?

> Wanted to put my C++ skills to the test, and also to create a decent PDF viewer for my daily use.

# Dependencies
	
1. Qt6
2. Poppler
